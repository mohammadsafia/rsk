## Custom Trigger Children

Override the default `+{count}` text by passing children to `CountTooltip.Trigger`.

<div className="flex items-center gap-2">
  <span className="text-sm">Documents</span>

  <CountTooltip count={2}>
    <CountTooltip.Trigger>View all</CountTooltip.Trigger>
    <CountTooltip.Item icon={<FileText size={14} />}>report_v1.pdf</CountTooltip.Item>
    <CountTooltip.Item icon={<FileText size={14} />}>report_v2.pdf</CountTooltip.Item>
  </CountTooltip>
</div>

## Custom Trigger Element

Skip `CountTooltip.Trigger` to use any element as the trigger. First child becomes the trigger, rest become content.

<div className="flex items-center gap-2">
  <span className="text-sm">report_v1.pdf</span>

  <CountTooltip count={2}>
    <Button variant="ghost" size="sm">+2 files</Button>
    <CountTooltip.Item icon={<FileText size={14} />}>report_v2.pdf</CountTooltip.Item>
    <CountTooltip.Item icon={<FileText size={14} />}>report_v3.pdf</CountTooltip.Item>
  </CountTooltip>
</div>

## Imports

import { CountTooltip } from '@components/shared';
