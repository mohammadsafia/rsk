## Variants

Bare names use the filled-light pattern (`bg-X-200 text-foreground border-X/50 [&>svg]:text-X`) — soft prominence, neutral readable text, colored border + icon for tone signal. Mirrors `Toast` + `Chip` conventions.

| Variant | Use case |
|---|---|
| `default` | Neutral message (muted bg) |
| `info` | Informational message (primary tone) |
| `success` | Successful action confirmation |
| `warning` | Caution or non-fatal issue |
| `destructive` | Error or fatal condition |
| `outline` | Outline-only primary alert (legacy / no-fill needs) |
| `outline-info` | Plain background informational alert (legacy) |
| `outline-destructive` | Outline-only error alert (legacy / less prominent) |
| `outline-success` | Outline-only success alert (less prominent than filled) |
