## Code

```tsx

<FormattedNumber showStepButtons min={0} max={100} stepper={5} />

// Currency formatting
<FormattedNumber prefix="$" thousandSeparator decimalScale={2} />

// With suffix
<FormattedNumber suffix="%" max={100} />
```
