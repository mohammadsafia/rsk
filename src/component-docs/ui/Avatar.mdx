## Code

```tsx
// With Image
<Avatar>
  <Avatar.Image src="..." alt="User" />
  <Avatar.Fallback>CN</Avatar.Fallback>
</Avatar>

// Custom Size
<Avatar className="h-12 w-12">
  <Avatar.Image src="..." alt="User" />
  <Avatar.Fallback>LG</Avatar.Fallback>
</Avatar>

// Icon Fallback
<Avatar>
  <Avatar.Image alt="User" />
  <Avatar.Fallback>
    <User />
  </Avatar.Fallback>
</Avatar>
```
