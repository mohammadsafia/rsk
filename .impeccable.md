# Design Context — RSK (React Starter Kit)

## Target audience
Frontend developers and teams evaluating or cloning RSK, a production-grade React + TypeScript + Vite starter template. Technical, design-aware, value DX and consistency.

## Use cases
- Landing page: quickly understand what the starter offers (component library, forms, tables, auth, i18n, DTO generator, MDX docs) and get started.
- App shell (authenticated): a dashboard with sidebar + header that teams build on.
- Reference implementation of the design system.

## Brand personality / tone
Modern, refined, confident — "crafted for engineers who value DX." Editorial-tech: precise and clean with a distinctive point of view. Not playful, not corporate-bland, not neon.

## Visual system
- **Palette:** Ocean Blue — cool slate neutrals dominant, blue (`#2563EB` / `--primary`) as a sharp, sparing accent. Full light + dark themes in OKLCH (`src/index.css`).
- **Typography:** Display = **Fraunces** (characterful editorial serif, optical sizing). Body = **DM Sans** (clean grotesque). Both loaded via Google Fonts in `src/index.css`.
- **Avoid (AI slop):** purple/blue gradients, glassmorphism, gradient text, neon-on-dark, identical repeated card grids, monospace-as-"technical".
